# PVP MockCET - High-Fidelity Mock Examination Platform

A premium Flutter application designed to simulate the MHT-CET (Maharashtra Health and Technical Common Entrance Test) experience. This platform features a role-based system for both students and administrators, with a focus on modern UI/UX and seamless performance.

## 🚀 Key Features

### 👨‍🎓 Student Portal
- **Secure Authentication**: Branded login system with roles.
- **Mock Test Interface**: Official MHT-CET pattern with:
    - 3-hour countdown timer.
    - Subject-wise question palette (Physics, Chemistry, Mathematics).
    - Question marking and review system.
- **Detailed Instructions**: Comprehensive test guidelines and marking schemes.
- **Consolidated Profile**: Unified view of personal, academic, and test history data.
- **Interactive Results**: Instant scoring and performance breakdown after every test.

### 🛡️ Admin Dashboard
- **Metric Tracking**: Real-time overview of active tests, students, and announcements.
- **Student Management**: List and monitor registered test-takers.
- **Question Management**: Interface for overseeing and updating mock test questions.
- **Broadcast System**: Send global announcements to all students.

## 🛠️ Technology Stack
- **Framework**: [Flutter](https://flutter.dev/) (Cross-platform)
- **Language**: [Dart](https://dart.dev/)
- **State Management**: [Provider](https://pub.dev/packages/provider)
- **Design System**: Material 3 with custom high-fidelity aesthetics.
- **Navigation**: Named routes with advanced safety fallbacks.

## 🏁 Getting Started

### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install)
- [Dart SDK](https://dart.dev/get-dart)
- An IDE (VS Code or Android Studio)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/mock-cet-demo.git
   ```
2. Navigate to the project directory:
   ```bash
   cd mock-cet-demo
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the application:
   ```bash
   flutter run
   ```
